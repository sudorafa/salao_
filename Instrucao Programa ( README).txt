Executa a script de banco para poder iniciar o uso do software.
Contatar um administrador ou criar um usuário administrador para utilizar o software.
Ao gerar relatório um arquivo de excel é criado no DISCO LOCAL C:\ 
Com os nomes: RelatorioFuncionario e RelatorioBalancoDiario.
Caso ocorrer algum erro verificar ao gerar o relatório verificar o caminho indicado no software em PersistenciaFuncionario e PersistenciaVenda.
